# Backend Setup (Node.js)

Navigate to the backend directory:

cd server

# Install dependencies:

npm install

# Run the backend server:

npm start

# Frontend Setup (React)

Navigate to the frontend directory:

frontend

# Install dependencies:

npm install

# Run the frontend server:

npm run dev
